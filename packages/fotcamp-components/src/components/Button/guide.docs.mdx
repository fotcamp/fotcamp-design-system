# Button Tokens
Button Tokens Guide
## Rule

```plain
seed_button_{properties}_{value}_{theme}
```

## Size

properties: size - \*

```css
seed_button_size_padding_x_large: 24px;
seed_button_size_padding_y_large: 16px;
seed_button_size_fontsize_large: 16px;
seed_button_size_height_large: 48px;

seed_button_size_padding_x_medium: 20px;
seed_button_size_padding_y_medium: 12px;
seed_button_size_font_size_medium: 14px;
seed_button_size_height_medium: 40px;

seed_button_size_padding_x_small: 16px;
seed_button_size_padding_y_small: 8px;
seed_button_size_font_size_small: 14px;
seed_button_size_height_small: 32px;
```

## Style

properties: style

```css
seed_button_style_fill: fill;
seed_button_style_outline: outline;
seed_button_style_flat: flat;
```

## Display

properties: display

```css
seed_button_display_inline: inline;
seed_button_display_block: block;
seed_button_display_full: full;
```
